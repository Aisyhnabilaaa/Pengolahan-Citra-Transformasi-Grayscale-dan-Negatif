# Pengolahan Citra: Transformasi Grayscale dan Negatif

Proyek ini menerapkan teknik pemrosesan citra untuk mengonversi gambar berwarna menjadi grayscale dan membuat gambar negatif.

## Fitur

- Konversi gambar berwarna ke grayscale dengan skala warna yang disesuaikan.
- Membuat citra negatif dari gambar.

## Instalasi

1. Clone repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependensi:
   ```bash
   pip install numpy matplotlib Pillow opencv-python
   ```

## Penggunaan

1. Pastikan file gambar (`Lena.png` dan `mammogram.tif`) berada di direktori proyek.
2. Jalankan script:
   ```bash
   python image_processing.py
   ```

## Dependensi

- `numpy`
- `matplotlib`
- `Pillow`
- `opencv-python`
